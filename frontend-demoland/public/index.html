🛡️ Security Features
🔒 AES-256-GCM Encryption
Military-grade encryption for all stored seed phrases and private keys. Keys never leave your device.
🧩 Shamir Secret Sharing
Split your master recovery key into 5 shares — any 3 can reconstruct. Distribute to trusted parties.
🔑 Hardware Key Binding
Optionally bind vault access to a hardware security key (YubiKey, Trezor, NitroKey) for 2FA.
🌐 Zero-Knowledge Proof
Prove vault ownership without revealing contents. Powered by Midnight blockchain zk-proofs.
